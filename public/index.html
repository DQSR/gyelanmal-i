실시간 변수, 실시간 리스트가 엔트리에 공식적으로 추가되었으니 해당 기능을 사용하시길 바랍니다.













계란말이

계란말이는 엔트리에서 실시간 변수 및 리스트를 사용할 수 있도록 만들어주는 비공식 확장 기능입니다!

쉬운 사용

'계란말이' 변수 하나만 추가하면
사용할 준비가 끝납니다.

빠른 속도

지리적으로 가까운 서버를 사용해
속도가 빠릅니다.

안정적 저장

NoSQL DB인 Firebase의 Firestore에
정보를 저장합니다.

설치 스크립트

      $.get('https://gyelanmal-i.web.app/index.prod.js')

어떻게 사용하나요?

계란말이 작품 만들기 >

1. '계란말이'라는 이름의 변수를 만듭니다.

2. 실시간 변수/리스트의
접두어로 사용될 단어를
'계란말이' 변수의 값으로 정하세요.

3. 실시간 변수/리스트로 사용할 변수/리스트의
이름을 앞에서 정한 접두어로 시작하도록
정하세요. 공유 변수/리스트로 사용을 체크 해야합니다.

계란말이 설치하기 >

'F12'를 눌러 개발자도구를 연 뒤,
콘솔(console)에 설치 스크립트를 입력하세요.

자주 묻는 질문

Q. 실시간 리스트가 작동하지 않아요! 어떻게 해야 하나요?
A. 기술적 한계로 설치 스크립트를 실행할 때 작품에 리스트가
하나도 추가되어 있지 않다면 실시간 리스트가 작동하지 않습니다.
리스트를 추가한 후 다시 설치 스크립트를 실행시켜주세요.
Q. 사용자가 계란말이를 설치했는지 어떻게 알 수 있나요?
A. 사용자가 계란말이를 설치했다면 '계란말이' 변수의 값이 '설치됨'으로 바뀝니다!
Q. 버그가 있어요!
A. Github Repo에 issue를 생성해주세요!
